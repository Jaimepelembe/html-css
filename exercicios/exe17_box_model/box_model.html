Propriedades das caixas

Usando short hands, as propriedades devem ser aplicadas no sentido horario começando de cima(top->right->bottom->left).

Width

Comprimento da caixa

Height

Altura da caixa

Border

Linha que circunda o contéudo da caixa

Padding

Espaço entre o conteúdo e a borda

Margin

Espaço que fica fora da borda. A propriedade margin auto centraliza o conteúdo da caixa.

Outline

Tracejado que fica fora do conteúdo

Tipos de caixas

Box level

Inicia em uma nova linha e ocupa todo o espaço horizontal; o conteúdo que vem depois continua numa nova linha.

Inline level

Se inicia na mesma linha, só ocupa o espaço necessário para o conteúdo.

Agrupamento das tags

Em html5 podemos agrupar as tags no header(cabeçalho), no main(conteúdo principal), no footer(rodapé), nav(barras de navegação), section(secção), article(artigo), aside

Cabeçalho

Normalmente uma grande faixa na parte superior com um grande título e/ou logotipo. É aí onde as principais informações comuns sobre um site geramlente ficam.


Exemplos

  • Amazon
  • Disney
  • Hostnet
  • Hotmart

Barra de navegação

Possui links para as principais seções do site; geralmente representado por botões de menu, links ou guias.
Existem dois tipos de barras de navegação, as horizontais e as verticais.

Exemplos

  • Barra horizontal
  • Barra vertical
