Tipografia

É uma arte que transmite sentimentos e emoções, ela surgiu da necessidade de ter letras mais faceis de ler e que aproveitassem melhor o espaço da página.

Principais conceitos

Os principais conceitos de tipografia são glifo, fonte, familia de fonte, e categorias de fonte.

Glifo

É uma letra de determinada fonte.

Fonte

É o conjunto de todos os glifos de uma familia. A fonte padrão nos navegadores é a Times New Roman.

As fontes mais usadas na Web são:

  • Arial (sans-serif)

    É amplamente utilizada na Web e na impressão de medias; ela está disponível para a maioria dos sistemas operacionais.

  • Verdana (sans-serif)

    É uma fonte muito popular, por ser de facíl leitura mesmo em tamanhos de fonte menores.

  • Tahoma (sans-serif)

    Ela tém pouco espaço entre os caracteres.

  • Trebuchet MS (sans-serif)

    Essa fonte deve ser utilizada cuidadosamente porque não é suportada por todos os sistemas operacionais moveis.

  • Times New Roman (serif)

    É uma das fontes mais utilizadas no mundo, ela tem uma aparência profissional e é utilizada em muitos jornais e websites.

  • Georgia (serif)

    Essa fonte tem uma legibilidade muito boa em diferentes tamanhos de fontes, sendo uma boa opção para o design responsivo em telemóveis.

  • Garamond (serif)

    É uma fonte classica utilizada em muitos livros impressos, ela tem uma aparência antiga e boa legibilidade.

  • Courier New (monospace)

    É a fonte monoespaçada(serif) mais utilizada, sendo aplicada em muitos provedores de email, telas de cinema,etc.

  • Brush Script MT (cursive)

    É uma fonte com escrita engraçada, ela é elegante e sofisticada, mas pode ser de dificil leitura; por isso deve ser usada com cautela.

Família

São as várias formas de representar um glifo(normal, negrito,italico,etc).

Categorias

Serifadas

São fontes que possuem serifas, elas servem para criar uma linha imaginária que nosso cerébro percebe para facilitar a leitura.

Sans-Serif

São fontes sem serifas, elas são ideias para textos muitos longos em telas.

Monoespaçadas

São fontes que possuem o mesmo espaçamento lateral, elas podem ter ou não serifas.

Handwring

São fontes que simulam a escrita humana.

Display

São fontes que transmitem o sentimento de diversão e comemoração.

Tamanho das fontes

As fontes podem ser medidas usando medidas absolutas ou medidas relaticas. As medidas padrão que devem ser usadas são pixel e em.

Medidas absolutas

  • cm
  • mm
  • in
  • px
  • pt(pontos)
  • pc

Essas medidas podem apresentar a informação de formas diferentes, dependendo do tamanho da tela.

Medidas relativas

  • em (tamanho letra M)
  • ex (..letra X)
  • rem
  • vw (tamanho da tela)
  • vh (altura da tela)

O tamanho padrão da fonte no navegador é 16px. Geralmente 16px=1em.

Shorthand

Um shorthand é uma simplificação de vários comandos em um único, para simplificar a sua declaração.
Para as fonte temos o shorthand font que tem a seguinte sintaxe: font: font-style -> font-weight -> font-size -> font-family .

Exemplo de aplicação

font: italic bolder 3em Verdana, 'sans-serif';

Captura de fontes

As fontes podem ser capturadas em páginas Web ou em imagens.

Fontes de páginas Web

Para capturar determinada fonte de uma página usamos a extensão do chrome fonts ninja.

Fontes de imagem

Para capturar as fontes em imagens usamos os sites whatfontis, fontsquirrel, myfonts

Alinhamento de textos

O alinhamento do conteúdo da página é um aspecto importante para a legibilidade e aparência geral de uma página da web. Em CSS podemos definir o alinhamento vertical e horizontal por meio de propriedades como text-align, display e flexbox.

Tex-align

Essa tag permite alinhar o nosso texto para a esquerda, direita, para o meio(center). Podemos ainda aplicar o text Indent que dá uma identação ao texto.
