Alinhamento de textos

O alinhamento do conteúdo da página é um aspecto importante para a legibilidade e aparência geral de uma página da web. Em CSS podemos definir o alinhamento vertical e horizontal por meio de propriedades como text-align, display e flexbox.

Tex-align

Essa tag permite alinhar o nosso texto para a esquerda, direita, para o meio(center). Podemos ainda aplicar o text Indent que dá uma identação ao texto.
