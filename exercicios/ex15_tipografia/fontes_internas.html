Fontes internas

Podemos adicionar fontes ao nosso site, baixando as para o nosso computador. Para adicionar essas fontes ao nosso computador usamos a tag font-face que tem como parâmetros a família, a url da fonte e o formato do arquivo.
os formatos das fontes são opentype(otf), truetype(ttf), embedded-opentype, truetype-aat(Apple Advanced Typography) e svg.

Captura de fontes

As fontes podem ser capturadas em páginas Web ou em imagens.

Fontes de páginas Web

Para capturar determinada fonte de uma página usamos a extensão do chrome fonts ninja.

Fontes de imagem

Para capturar as fontes em imagens usamos os sites whatfontis, fontsquirrel, myfonts
