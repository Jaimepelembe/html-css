Mídias

É muito importante possuir um site com responsividade pois isso melhora a experiência do usuario fazendo com que ele fique mais tempo no site. As principais mídias trabalhadas na internet são imagens, aúdio e video.

Imagens dinâmicas

Uma imagem dinâmica é aquela que se ajusta ao tamanho da tela.
Para criar uma imagem dinâmica deve-se gerar tamanhos diferentes da mesma imagem e usar as tag <picture> e <source> para fazer o navegador carregar a imagem certa para cada dispositivo.
A tag picture é um contentor de sources que permite ao navegador carregar a imagem certa de acordo com o tamanho da tela, densidade de pixels,etc; dentro da tag deve ter uma imagem padrão que deve ser inserida usando a tag <img> essa imagem deve ser a maior de todas.
A tag <source> permite especificar múltiplas alternativas de médias; as imagens devem ser colocadas em ordem decrescente de tamanho a partir da imagem padrão;

Os principais parâmetros dessa tag source

  • type: indica o tipo da média;
  • srcset: indica o caminho da imagem a ser carregada;
  • media: indica o tamanho de tela máximo a ser considerado para carregar a imagem;
    o tamanho máximo=tamanho da foto maior + 50, para evitar a barra de rolagem.

Principais tamanhos de imagem

Os principais tamanhos de imagens usados para web são:

  • 144p: 256x144
  • 240p: 320x240
  • 480p: 848x480
  • HD: 1280x720
  • Full HD: 1920x1080
  • 4K: 3840x2160
  • 8K: 7680x4320

Áudio

O áudio é um recurso muito importante pois pode ser usado para apresentar várias informações ao usuario como músicas, podcasts, hinos, etc.
O áudio utilizado para apresentação de conteúdo deve ser de baixa qualidade para evitar arquivos muito grandes.
Para inserir áudio usamos a tag <audio> que é um contetor de áudios;

Os principais parâmetros da tag audio são:

  • Preload: indica se o áudio será pre-carregado ou não; os valores desse parâmetro são:
    • metadata - carrega as informações do arquivo;
    • none - não carrega nenhuma informação do arquivo;
    • auto(padrão) - carrega todo o arquivo;
  • controls: apresenta o player na tela;
  • autoplay: toca o áudio automaticamente logo que a página é carregada;
  • loop: repete o áudio infinitamente;

Componente source:src

Serve para colocar diferentes formatos do mesmo áudio dentro da tag audio; deve-se criar um parágrafo dentro da tag audio para permitir o download do arquivo mp3 no caso do navegador não suportar nenhum dos formatos disponibilizados.

Os principais formatos de áudio são:

  • mp3(mpeg): é o padrão
  • ogg
  • wav: é muito pesado

Vídeos

Os videos são um recurso muito importante na internet pois conseguem estimular vários sentidos e captar a atenção do usuario por muito tempo.
Para inserir videos internos usamos a tag <video>;

Os principais parâmetros da tag video:

  • width: indica o comprimento da janela do video;
  • height: indica a altura da janela do video;
  • poster: coloca uma imagem como a capa do video;
  • controls: exibe o player do video;
  • autoplay: reproduz o video automaticamente logo que a página é carregada;

Principais formatos de video:

Os principais formatos de videos suportados pelos navegadores são:

  • mp4 e mp4v
  • webm
  • ogv (ogg)

Hospedagem de videos

Os videos podem ser hospedados internamente ou externamente.

Hospedagem interna

Os videos são mantidos no nosso servidor.
Esse tipo de hospedagem traz alguns problemas como muito consumo de banda larga a carretando em custos adicinais, torna o site lento e pode ocorrer incompatibilidade com alguns navegadores por causa do formato do video; para evitar problemas de incompatibilidade deve-se gerar vários formatos do mesmo video.

Hospedagem externa

Os videos são mantidos em servidores externos.
Esse tipo de hospedagem não consome a minha banda, os principais serviços de são o YouTube e o vimeo

hospedagem no YouTube:
