Estilos

CSS (Folha de Estilo em Cascata) é o código que você usa para dar estilo à sua página Web.Existem três tecnicas para dar estilos a uma página web, elas são estilos inline, internal e external./p>

Estilos inline

São aqueles usados dentro de uma tag html;
esse estilo não é muito usado porque polui o código e não permite fazer alterações simultânias em tags iguais;
