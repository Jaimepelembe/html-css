Listas ordenadas

Para criar listas ordenadas usamos a tag ol, que significa ordered lists e tem como parâmetros o tipo ( 1, A, a, I, i ) e o começo da lista.
Para adicionar um elemento a lista usamos a tag li que significa list item

Lista de definições

Para criar a lista usamos as tags dl, dt e dd, onde dl é a lista, dt refere-se ao termo e dd refere-se a definição.
