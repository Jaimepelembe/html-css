Selectores Personalizados

Class

Usamos a tag class para criar um selector de classes que é equivalente a um ponto(.) em css.
A class deve ser nomeada de acordo com a sua funcionalidade.

Id

Usamos a tag id para criar um selector de id que é equivalente a um asterisco(#) em css.

Exemplo de aplicação

Diferenças entre class e id

O id só deve ser usado em um único elemento do documento html, enquanto que a class pode ser usada em um ou mais elementos e o mesmo elemento pode ter várias classes.

Pseudo-classe

É um selector em CSS que permite selecionar elementos que não estão directamente presentes no código HTML, mas sim em um estado ou em um contexto específico.
As pseudo-classes são usadas para aplicar estilos a elementos HTML com base em acções do usuário, como hover, focus ou em seu estado actual, como visited ou active.

Hover

É usado para aplicar um estilo a um elemento quando o cursor do mouse é colocado sobre ele.
Para aplicar a pseudo-classe hover em um elemento com determinado id ou classe, deve-se adicionar a pseudo-class ':hover' após o selector de id ou class.

Pseudo-elementos

São usados para adicionar conteúdo adicinal a um elemento HTML ou para aplicar estilos a uma parte espcífica de um elemento.
